# ai-dev-workflow-template
aiとgithubを使う開発の個人的テンプレ用
## What is this?
AI（Codex / Gemini / Jules）を主戦力にした
GitHub Issue 駆動開発テンプレート。

## Core rule
1 Issue = 1 AI implementation = 1 PR

## How to use
1. Use this template
2. Run init script (see scripts/)
3. Create Issue using form
4. Add `assign-to-jules`
5. Review PR
