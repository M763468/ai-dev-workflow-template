# PROMPTS

このファイルは、AIエージェントに渡すプロンプトの置き場です。

## 使い方（例）

- `docs/PROMPTS.md` に「定型プロンプト」を蓄積する
- Issue本文は「要件の契約書」として厳密に書く
- プロンプトは「補助」として使い、Issueと矛盾させない

## 例: PRレビュー依頼プロンプト（雛形）

```text
あなたはリポジトリのレビュアーです。以下の観点でPRをレビューしてください:
- Acceptance Criteria を満たしているか
- Scope (In / Out) を逸脱していないか
- 破壊的変更がないか
- テスト/ドキュメント更新が妥当か
```
